Arquitectura moderna para garantizar escalabilidad y alta disponibilidad.

Infraestructura de Cómputo

App Server con Azure Virtual Machines para aplicaciones empresariales.

  • Servicio: Azure Virtual Machines (Serie D o E)
  • OS Destino: Windows Server 2022/2025
  • Soporte: Extendido y seguridad avanzada
  • Método: Re-provisionamiento (Clean install)

Capa de Datos - Opción A (IaaS)

SQL Server on Azure VM con control total del sistema operativo.

  • Servicio: SQL Server on Azure VM
  • Control total del Sistema Operativo
  • Configuración personalizada de base de datos
  • Ideal para: Cargas de trabajo tradicionales

Capa de Datos - Opción B (PaaS)

Azure SQL Database como servicio administrado con máxima escalabilidad.

  • Servicio: Azure SQL Database
  • Servicio administrado por Microsoft
  • Modelos: Serverless o Provisioned
  • Escalabilidad automática y alta disponibilidad

Arquitectura de Referencia

Solución basada en servicios de Microsoft Azure que garantiza un entorno robusto, seguro y gestionable.
